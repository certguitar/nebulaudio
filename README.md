# 🌌 Nebulaudio

Música original con IA para creadores digitales — en segundos, sin derechos de autor.

> ✨ **Beta abierta** — Generación real próximamente.  
> Este repositorio contiene el frontend estático de Nebulaudio: una interfaz moderna para generar música e instrumentales con IA, enfocada en YouTubers, podcasters y artistas emergentes.

---

## 🎯 ¿Qué ofrece Nebulaudio?

- ✅ **Música personalizada**: elige estilo, emoción, instrumentos y tonalidad  
- ✅ **Modo cantante (Studio)**: genera una canción con voz IA + instrumental limpio para grabar tú  
- ✅ **Previsualización honesta**: escucha y ve la partitura antes de pagar  
- ✅ **Sin jerga técnica**: diseñado para no músicos

## 🚀 Cómo probarlo localmente

1. Clona o descarga este repositorio  
2. Abre `index.html` en tu navegador (Chrome, Firefox, etc.)  
   → ¡Listo! No requiere servidor ni instalación.

Opcional (mejor experiencia):
- Instala la extensión **[Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)** en VS Code  
- Haz clic derecho en `index.html` → *Open with Live Server*
